Задание 2

Консольное приложение с объектно-ориентированной архитектурой, выводящее характеристики заданной геометрической фигуры.

* Входные данные:
Тип фигуры и ее параметры читаются из файла. Для разных типов фигур параметры отличаются.
Первой строкой в файле идет код типа фигуры. Второй строкой – параметры фигуры через пробел.
Входные параметры задаются аргументами командной строки (-f вывод в файл, -c вывод в консоль).


* Пример:
CIRCLE
5


* Выходные данные:
Результатом работы приложения являются вычисленные характеристики фигуры. Для разных фигур характеристики отличаются, 
так же они зависят от введенных параметров фигуры. Результат выводится в файл или в консоль, 
это конфигурируется через аргумент командной строки.
