Задание 4

Программа вычисляет сходящийся ряд обратных степеней числа 2 (сила двух)

* Вычисляет сложную функцию для каждого целого числа от 1 до N, N – большое число, например, 10 000 000
   N – ввод с консоли


* Выводит на экран сумму посчитанных значений


* Поскольку N – большое, вычисления разбиваются на несколько частей и каждая часть вычисляется в отдельном потоке параллельно.


* В приложении используется логирование.


* Задача выполнена с использованием java.concurrency
